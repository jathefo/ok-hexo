
## Getting started
Install dependencies:
``` bash
$ git clone https://github.com/jathefo/ok-hexo.git
$ cd ok-hexo
$ npm install
$ npm install -g hexo
``` 
Generate:
``` bash
$ hexo generate
``` 
Run server:
``` bash
$ hexo server
``` 
Deploy to remoute sites:
``` bash
$ hexo deploy
```
